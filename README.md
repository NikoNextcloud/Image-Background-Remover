# Чист Фон

Уеб приложение за автоматично премахване на фон от JPG, PNG и WebP изображения. Обработването се извършва директно в браузъра и резултатът се сваля като прозрачен PNG.

## Публикуване в GitHub Pages

1. Създайте ново публично GitHub repository.
2. Качете **съдържанието на тази папка** в основната директория на repository-то.
3. В GitHub отворете **Settings → Pages**.
4. При **Build and deployment → Source** изберете **GitHub Actions**.
5. Изчакайте действието `Publish to GitHub Pages` да завърши със зелена отметка.

Готовият адрес ще бъде `https://ВАШЕТО-ИМЕ.github.io/ИМЕ-НА-REPOSITORY/`.

## Локално стартиране

Необходими са Node.js 20 или по-нова версия:

```bash
npm install
npm run dev
```
